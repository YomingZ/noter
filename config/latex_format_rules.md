# LaTeX 数学公式格式规范（Obsidian 版）

> **用途**：每次让 AI 生成 Obsidian 笔记前，将此文件内容（或关键部分）发送给 AI，确保公式格式正确。
> **核心原则**：Obsidian 仅识别 `$...$`（行内）和 `$$...$$`（显示），**不识别** `\(...\)` 和 `\[...\]`。

---

## 一、分隔符规则（最重要的规则）

### ✅ 正确用法

| 类型 | 格式 | 示例 |
|------|------|------|
| **行内公式** | `$...$` | `$E = mc^2$` → $E = mc^2$ |
| **显示公式** | `$$...$$`（独占一行） | `$$\frac{1}{2}$$` → 独立显示 |

### ❌ 绝对禁止

```
禁止1：\(...\) 或 \[...\]
  ❌ \(E = mc^2\)
  ✅ $E = mc^2$

禁止2：双重包裹 $...$ 内部嵌套 \(...\)
  ❌ \($\hat{$O}$\)
  ✅ $\hat{O}$

禁止3：$$...$$ 内部混入多余 $
  ❌ $$\Psi^$*(x,t)$$
  ✅ $$\Psi^*(x,t)$$

禁止4：反引号包裹公式
  ❌ `($\int \Psi dx$)`
  ✅ $(\int \Psi dx)$
```

## 二、LaTeX 命令书写规则

### ✅ 正确写法

```
\frac{a}{b}    ← 分式
\hat{O}        ← 算符帽子
\partial       ← 偏微分
\Psi           ← 波函数
\hbar          ← 约化普朗克常数
\lambda        ← 拉姆达
\sigma         ← 西格玛
\infty         ← 无穷
\int           ← 积分
\sum           ← 求和
\exp           ← 指数函数
\left(  \right) ← 自适应括号
```

### ❌ 绝对禁止

```
禁止1：命令前加 $
  ❌ $\frac{a}{b}$、$\hat{O}$、$\Psi$
  ✅ \frac{a}{b}、\hat{O}、\Psi

禁止2：命令中使用 \$ 转义
  ❌ $\hbar\$frac{...} → 命令断裂
  ✅ \hbar\frac{...}

禁止3：参数中插入 $
  ❌ {$2}、\hat{$H}、\partial^$2
  ✅ {2}、\hat{H}、\partial^2
```

## 三、大括号闭合规则

### ❌ 常见错误

```
❌ \frac{1}{(2\pi\sigma^2)^{1/4} \exp(...)
    → ^{1/4} 后缺少 }，导致 \exp 被吸入分母

✅ \frac{1}{(2\pi\sigma^2)^{1/4}} \exp(...)
    → ^{1/4}} 后分母正常闭合
```

**检查方法**：`{` 和 `}` 的数量必须相等。

### ✅ 嵌套大括号的计数

```
{ 外层 { 内层 } 内层 } 外层    ← 正确（2个 {，2个 }）
{ 外层 { 内层 } 外层           ← 错误（2个 {，1个 }）
```

## 四、行内 vs 显示公式的判断标准

### 简单公式 → 用 $...$

```
$E = mc^2$
$\Psi(x,t)$
$\hat{O}$
$\lambda_n$
$\frac{h}{2\pi}$        ← 单个 \frac 且在行内
$x_0$、$\sigma$、$p_0$
```

### 复杂公式 → 用 $$...$$（独占一行）

```
$$
\hat{H} = -\frac{\hbar^2}{2m}\frac{\partial^2}{\partial x^2}
$$
← 多个 \frac、\int、\sum 等

$$
[\hat{A}, \hat{B}] = \hat{A}\hat{B} - \hat{B}\hat{A} = 0
$$
← 长度超过 50 字符或 \ 命令超过 4 个

$$
\int_{-\infty}^{\infty} \Psi^*(x,t) \hat{O} \Psi(x,t) dx
$$
← 包含 \int、\sum、\prod 等大型运算符
```

### 特别规则：括号内的公式

```
❌ （$$\int |\Psi|^2 dx = \infty$$）
    → $$ 独占一行，导致括号与公式分离

✅ （$\int |\Psi|^2 dx = \infty$）
    → 用行内 $...$，括号自然包裹公式
```

## 五、其他格式要求

### 绝对值/范数

```
❌ \|\Psi(x,t)\|    ← 双竖线
✅ |\Psi(x,t)|      ← 单竖线
```

### 多行公式

```latex
$$
\begin{align}
E &= \frac{p^2}{2m} \\
p &= \hbar k
\end{align}
$$
```

### 矩阵

```latex
$$
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$
```

## 六、完整错误对照表

| # | ❌ 错误写法 | ✅ 正确写法 | 错误类型 |
|---|-----------|-----------|---------|
| 1 | `\($\hbar = \frac{h}{2\pi}$\)` | `$\hbar = \frac{h}{2\pi}$` | 双重分隔符污染 |
| 2 | `\(\int \Psi^2 dx = \infty\)` | `$\int \Psi^2 dx = \infty$` | 不支持 `\(` |
| 3 | `\(\hat{H} = -\frac{\hbar^2}{2m}\)` | `$$\hat{H} = -\frac{\hbar^2}{2m}$$` | 不支持 `\(` + 复杂公式 |
| 4 | `$$\Psi^$*(x,t)$$` | `$$\Psi^*(x,t)$$` | 显示公式中混入 $ |
| 5 | `\frac{1}{(2\pi\sigma^2)^{1/4} \exp` | `\frac{1}{(2\pi\sigma^2)^{1/4}} \exp` | 大括号不匹配 |
| 6 | `$\sigma\$)` | `$\sigma$` | `\$` 破坏 `\)` |
| 7 | `$\hbar\$frac{...}` | `\hbar\frac{...}` | `\$` 破坏命令 |
| 8 | `\(p\), \(E\)` | `$p$, $E$` | 不支持 `\(` |
| 9 | `\|\Psi\|` | `|\Psi|` | 双竖线错误 |
| 10 | `` `($\int dx$)` `` | `$(\int dx)$` | 反引号包裹 |

## 七、生成前的自检清单

在输出笔记前，请逐条检查：

- [ ] 所有公式分隔符只有 `$...$` 和 `$$...$$`，没有 `\(...\)` 或 `\[...\]`
- [ ] 所有 `$...$` 内部没有多余的 `$` 符号
- [ ] 所有 LaTeX 命令（`\frac`、`\hat`、`\Psi` 等）前面没有 `$`
- [ ] 所有大括号成对出现，`{` 和 `}` 数量相等
- [ ] 没有 `\$` 转义（在 LaTeX 公式中不需要转义 `$`）
- [ ] 简单/短公式用 `$...$`，复杂/长公式用 `$$...$$`
- [ ] 括号 `（...）` 内不使用 `$$` 显示公式，改用 `$`
- [ ] 没有反引号包裹的公式
- [ ] `\|` 替换为 `|`，`\|\|` 替换为 `|`
- [ ] `align`、`matrix` 等环境用 `$$...$$` 包裹
